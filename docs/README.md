# Dokumentasi NiagaBio

Folder ini berisi dokumentasi project, desain, setup, roadmap, dan catatan patch.

## File penting

- `PROJECT_STRUCTURE.md` — peta file/folder.
- `UPDATE_GUIDE.md` — urutan update dan checklist test.
- `SETUP_SUPABASE_DARI_NOL.md` — panduan setup Supabase.
- `DESAIN.md` — pedoman desain NiagaBio.
- `ROADMAP.md` — roadmap fitur dan prioritas.
- `patch-notes/` — catatan patch per tahap.

Mengubah file di folder ini tidak mengubah website production.
