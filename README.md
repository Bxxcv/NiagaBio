# NiagaBio

NiagaBio adalah website **link bio + katalog produk + checkout WhatsApp/QRIS manual** untuk seller, kreator, freelancer, dan UMKM.

Project ini dibuat dengan:

- HTML
- CSS
- JavaScript vanilla
- Bootstrap 5
- Bootstrap Icons
- Supabase Auth / Database / Storage

Tidak memakai React, jadi lebih cocok untuk editing dari HP.

## Mode Checkout

### WhatsApp

Pembeli klik beli lalu langsung diarahkan ke WhatsApp seller.

### QRIS Manual

Seller upload QRIS sendiri. Pembeli scan QRIS, upload bukti, lalu pesanan masuk dashboard seller. Seller klik **Paid** agar omset bertambah.
