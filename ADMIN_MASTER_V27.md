# NiagaBio v27 - Admin Master Professional Fix

Patch ini fokus memperbaiki Admin Master supaya tidak penuh, lebih responsif, dan lebih cocok sebagai panel owner platform.

## Perubahan utama

- Menu Order Platform dihapus dari Admin Master.
- Diganti menjadi menu Laporan Platform.
- Ringkasan admin dibuat responsif dengan KPI grid.
- Omset admin diganti menjadi estimasi pendapatan Premium dari request approved.
- Request Premium default menampilkan pending dulu.
- Request yang sudah approved/rejected bisa dibersihkan sekaligus.
- Request Premium sekarang tampil sebagai card, bukan tabel sempit.
- Admin bisa hapus request satu per satu.
- Laporan Platform punya export CSV user/request dan print laporan.

## Catatan

Order toko tetap ada di menu Pesanan milik seller/user. Admin Master tidak lagi mengelola order seller agar fokus pada user, request premium, pendapatan platform, dan setting.

Tidak perlu menjalankan SQL baru untuk patch ini.
