Kebijakan Privasi

Terakhir diperbarui: 2026

Data yang disimpan

NiagaBio menyimpan data yang kamu isi untuk menjalankan toko, seperti nama toko, email akun, produk, link, sosial media, gallery, QRIS checkout, order, dan bukti transfer yang kamu unggah.

Penggunaan data

Data dipakai untuk menampilkan halaman toko, memproses order manual, membantu admin mengecek request premium, dan menjaga keamanan akun.

Kontrol user

User bisa mengubah data toko dari dashboard. Untuk penghapusan akun atau data tertentu, hubungi admin NiagaBio.
