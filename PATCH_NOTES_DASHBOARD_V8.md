# NiagaBio Dashboard User v8

Patch ini fokus merapikan dashboard user tanpa mengubah SQL, Supabase config, atau logic admin.

## File yang diubah

- `dashboard.html`
- `assets/js/dashboard.js`
- `assets/css/main.css`

## Perubahan utama

- Hero dashboard lebih profesional.
- Link public toko otomatis berdasarkan username user.
- Tombol copy link public toko.
- Card paket Free/Premium lebih jelas.
- Statistik toko: produk, order, pending, omset paid.
- Progres setup toko: profil, produk, link/sosial, checkout.
- Limit plan: produk, link custom, sosial, gallery.
- Aksi cepat: produk, profil, checkout, tema.
- Pesanan terbaru lebih rapi.
- Tampilan mobile dashboard diperbaiki.

## Catatan

- Tidak ada perubahan database.
- Tidak ada perubahan `assets/js/config.js`.
- Tidak ada perubahan admin panel.
- Tidak ada perubahan RLS Supabase.
