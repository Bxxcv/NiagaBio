CARA FIX:

2. Hapus semua isi query lama.
3. Paste isi file: supabase/schema_clean_run_this.sql atau supabase/schema.sql dari zip ini.
4. Run dari baris pertama sampai akhir. Jangan run mulai dari tengah/line 860.
5. Kalau sudah pernah register admin, jalankan bootstrap admin yang ada di bagian bawah schema.
