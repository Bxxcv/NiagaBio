# Soft Launch Checklist NiagaBio

## Sebelum dibagikan

- [ ] `DEMO_MODE` sudah `false`
- [ ] Supabase Auth email provider aktif
- [ ] Confirm email dimatikan dulu untuk testing
- [ ] Admin Master bisa login
- [ ] QRIS Premium sudah diisi di Admin Master
- [ ] Maintenance mode mati
- [ ] Link demo toko bisa dibuka
- [ ] Checkout produk bisa dibuat
- [ ] Pesanan masuk ke dashboard seller
- [ ] Rekap penjualan terbaca
- [ ] Privacy, Terms, Refund bisa dibuka
- [ ] Favicon dan preview WhatsApp sudah muncul

## Test akun

- [ ] Akun admin
- [ ] Akun free
- [ ] Akun premium
- [ ] Visitor incognito untuk checkout

## Setelah dibagikan ke tester

Catat:

- Bug teknis
- Halaman yang membingungkan
- Teks yang kurang jelas
- Fitur yang paling dicari
- Alasan mereka mau / tidak mau upgrade

## Batas soft launch

Mulai dari 5–10 seller dulu. Jangan langsung promosi besar sebelum feedback pertama selesai.
